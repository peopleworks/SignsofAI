# Self-check (run this on every edit before returning it)

After rewriting a draft in Edit mode, verify your own output against this checklist. If any item fails,
fix it and re-check. The goal is subtraction without distortion.

## Fidelity (must all pass)
- [ ] **Same meaning.** No claim, fact, number, name, or citation was added, dropped, or altered.
- [ ] **Same language.** English stayed English; Spanish stayed Spanish. You did not translate.
- [ ] **Same voice.** It still sounds like the author, not like a different, "polished" writer.
- [ ] **Similar length.** You edited, you did not expand into a new essay or gut it to a stub.

## No new slop introduced (re-scan your own rewrite)
- [ ] No overused vocabulary crept in (delve, tapestry, leverage, robust … / sus análogos en español).
- [ ] No empty intensifiers (just, simply, actually, truly … / simplemente, realmente …).
- [ ] No filler phrases ("it's worth noting", "at the end of the day" / "cabe destacar", "al final del día").
- [ ] No rhetorical crutches (negative parallelism, throat-clearing, weasel attribution, hype, faux-insight).
- [ ] No copula avoidance ("serves as", "plays a crucial role" / "se erige como", "juega un papel crucial").
- [ ] No em-dash overuse; no emoji in headings; no mid-sentence bold.

## Rhythm (the tell most edits miss)
- [ ] Sentence lengths **vary** — at least one short, punchy sentence breaks up the long ones. If every
      sentence is a similar length, the rewrite still reads like a machine even with clean vocabulary.

## Honesty
- [ ] You did not invent a numeric score. If the user wants a number, you pointed them to the engine
      (`signsofai check`, the web app, or the `analyze_ai_writing` MCP tool) instead of guessing one.

A rewrite that passes fidelity but fails the rhythm check is not done. Vary the cadence, then return it.
